If sections appear empty:

Common Issues:

  • CORS Error: If opening from file:// protocol, use a local server instead
  • CSS not loading: Check the browser console for 404 errors
  • JavaScript disabled: Content should still be visible (animations won't work)
  • CDN blocked: Check internet connection or firewall settings

Quick Fix - Use a Local Server:

# Using Python 3
python3 -m http.server 8000

# Using PHP
php -S localhost:8000

# Using Node.js (npx)
npx http-server -p 8000

# Then open: http://localhost:8000
